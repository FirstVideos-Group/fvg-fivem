fvg-notify v1.0.0

FirstVideos Group – FiveM Értesítési Rendszer · Export dokumentáció & interaktív demo

Kliens exportok

exports['fvg-notify']:Notify(data)
Értesítést jelenít meg az aktuális kliens képernyőjén. Hívható bármely más script kliens oldaláról.
type
string – success · error · info · warning · police · money
title
string – Értesítés fejléce (opcionális)
message
string – Értesítés szövege
duration
number – Megjelenési idő ms-ben (alapért. 4000)
-- Lua példa (kliens script) exports['fvg-notify']:Notify({ type = 'success', title = 'Sikeres!', message = 'Az autót sikeresen beparkoltad.', duration = 4000 })

Szerver exportok

exports['fvg-notify']:NotifyPlayer(source, data)
Egyetlen játékosnak küld értesítést szerver oldalról.
-- Lua példa (szerver script) exports['fvg-notify']:NotifyPlayer(source, { type = 'money', title = 'Fizetés', message = 'Megkaptad a heti fizetésedet: $5,000' })
exports['fvg-notify']:NotifyAll(data)
Minden online játékosnak küld értesítést egyszerre.
-- Lua példa (szerver script) exports['fvg-notify']:NotifyAll({ type = 'info', title = 'Szerver', message = 'A szerver 5 perc múlva újraindul!' })
