exports['fvg-engine']:IsEngineOn() → boolean
Client
Visszaadja, hogy a helyi játékos járművének motora jelenleg jár-e.
Lua példa
-- fvg-vehicledamage, fvg-fuel stb. local running = exports['fvg-engine']:IsEngineOn() if not running then -- motor le van állítva, ne fogyasszon üzemanyagot end
exports['fvg-engine']:SetEngineOn(state, silent)
Client
Programozottan indítja el vagy állítja le a motor állapotát. A silent paraméterrel elnyomható a hangeffekt és az értesítés – hasznos automatikus műveleteknél. Frissíti az fvg-vehiclehud motor, sebesség és RPM moduljait.
ParaméterTípusLeírás
state boolean true = motorindítás, false = leállítás
silent boolean Ha true, nem játszik le hangot és nem küld értesítést (opcionális, alap: false)
Lua példa – fvg-vehicledamage
-- Motor leállítás kritikus sérülésnél (hangjelzés nélkül) if engineHealth <= 0 then exports['fvg-engine']:SetEngineOn(false, true) end -- Motor visszaindítás szerelés után exports['fvg-engine']:SetEngineOn(true, false)
exports['fvg-engine']:GetEngineState() → table
Client
Részletes állapotobjektumot ad vissza a motor aktuális állapotáról.
Visszatérési mezőTípusLeírás
engineOn boolean Jár-e a motor
inVehicle boolean Járműben van-e a játékos
isShutting boolean Folyamatban van-e a motor fokozatos leállítása
vehicleClass number Aktuális jármű GTA osztálya (-1 ha nincs jármű)
Lua példa
local state = exports['fvg-engine']:GetEngineState() -- state.engineOn = true/false -- state.inVehicle = true/false -- state.isShutting = false -- state.vehicleClass = 0 (pl. kompakt autó)
Szerver exportok
exports['fvg-engine']:GetPlayerEngineState(playerId) → boolean
Server
Szerver oldalról lekérdezi egy játékos járművének motor állapotát. Használható admin eszközökhöz, statisztikákhoz.
ParaméterTípusLeírás
playerIdnumberA játékos szerver ID-ja
Lua példa – szerver script
-- Szerver oldali motor lekérdezés local running = exports['fvg-engine']:GetPlayerEngineState(source) -- Szerver oldali kényszer leállítás (pl. rendőrségi EMP) -- fvg-engine/server/server.lua ForceEngine eseményen keresztül: TriggerServerEvent('fvg-engine:server:ForceEngine', targetPlayerId, false, false)
